It is a Chrome extension for printing JSON and JSONP.

Features
* Syntax highlighting
* Collapsible nodes
* Clickable URLs
* URL does not matter
* Inspect your json typing "json" in the console
* Options page with:
   - 8 themes
   - Max json size (200kb default)
   - Option to configure the font-size
   - Option to activate/deactivate "nodes always collapsed" (from 2ยบ level)
   - Option to activate/deactivate the header (timestamp + url)

Note
This extension might crash with other JSON highlighters/formatters, you may need to disable them
